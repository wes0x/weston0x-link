# @weston0x — link in bio

Public landing page for the WesLocks TikTok bio, hosted free on GitHub Pages.

**Live URL:** https://wes0x.github.io/weston0x-link/

## How to make the buttons work
Open [`index.html`](index.html) and replace each `href="#"` with your real link. There are 4:

| Button | Replace `href="#"` with |
|--------|------------------------|
| 🔒 Today's Free Pick | your **Discord invite** link |
| 💎 Join VIP — Full Card | your **Whop checkout** link (this is the payment step) |
| 📊 Track Record | your Discord `#track-record` or a proof link |
| 🎬 TikTok | your TikTok profile `https://tiktok.com/@weston0x` |

Each button has a `<!-- SWAP ... -->` comment right above it telling you which link goes there.

After editing: commit + push, and GitHub Pages redeploys the page automatically in ~1 minute.

> Note: this repo is **public** (GitHub Pages requires it on the free plan). It contains only the public bio page — the WesLocks strategy/playbook lives in the separate **private** `weslocks` repo.
